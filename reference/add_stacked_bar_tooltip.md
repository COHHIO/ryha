# Add tooltip to stacked bar chart

`add_stacked_bar_tooltip()` adds a tooltip to an `echarts4r` stacked bar
chart. The tooltip is triggered on the axis and displays values along
with their percentages.

## Usage

``` r
add_stacked_bar_tooltip(echart)
```

## Arguments

- echart:

  An `echarts4r` object representing the stacked bar chart.

## Value

An `echarts4r` object with the tooltip applied.
