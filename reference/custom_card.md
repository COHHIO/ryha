# Create a custom card

`custom_card()` generates a Bootstrap card with a specified height and
full-screen capability.

## Usage

``` r
custom_card(..., height = "490px")
```

## Arguments

- ...:

  Additional arguments passed to
  [`bslib::card()`](https://rstudio.github.io/bslib/reference/card.html)

- height:

  A character string specifying the card's height. Default is `"490px"`.

## Value

A [`bslib::card`](https://rstudio.github.io/bslib/reference/card.html)
object.
