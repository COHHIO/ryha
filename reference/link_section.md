# Create span with hyperlink to the HMIS Data Standards Manual

`link_section()` generates a span that references a section and has a
hyperlink to the HMIS Data Standards Manual. The link opens the manual
in a new browser tab.

## Usage

``` r
link_section(section)
```

## Arguments

- section:

  The name or number of the section to refer to.

## Value

HTML span with a referenced section and a hyperlink to the HMIS Data
Standards Manual.

## Examples

``` r
if (FALSE) { # \dontrun{
link_section("R4 Last Grade Completed")
} # }
```
